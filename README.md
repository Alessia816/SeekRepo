# SeekRepo
 This project recreates the home screen of the Seek by iNaturalist app, following Apple's Human Interface Guidelines to ensure a consistent and intuitive user experience. In addition to the UI work, the project includes functionality to request and manage the use's location in order to display nearby species.
 
 
 # Features
 
 - User Location Access: Requests permission to access the user's location and displays nearby species.
 
 - Category Filters: Allows filtering of species by different categories for easier browsing.
 
 - Community Link: Includes a link at the bottom of the page directing users to iNaturalist community website.
 
 - UI Accuracy: Designed to closely follow the Seek app's home screen layout and interactions.
 
 
 # Future improvements
 
 - Integration with real observation data.
 
 - Camera integration for capturing observations directly within the app.
 
 - Species recognition with CoreML to identify organisms from camera input.
 
 - Additional screens inspired by Seek.
